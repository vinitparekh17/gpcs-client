## Setup

1. Clone the repository:

   ```sh
   git clone https://github.com/vinitparekh17/openai-client.git
    ```

2. Install NPM packages:

   ```sh
   npm install
    ```

3. Run the project:

    ```sh
   npm run dev
    ```

4. Open the project in your browser:

   ```sh
   http://localhost:3000
    ```
